Numerical Methods in Cloth Simulation

This is a MATLAB simulation done for a computational physics course, which seeked to understand and implement the common numerical techniques used in cloth simulation. The relevant equations were taken directly from Baraff-Witkin's paper as well as Choi-Ko's paper. Various solving techniques including both explicit and semi-implicit schemes were used to solve the differential equations.
