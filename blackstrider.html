Black Strider

Black Strider is a 2D infinite runner game made with Unity Engine. The game uses open-sourced sprites (ex. Strider) for building animated assets. It leverages Unity's state machine behaviour heavily for gameplay controls. Basic 2D navigation mesh is also used for the robots.
